# 💖 CareConnect AI

## 🚀 Overview
CareConnect AI is an intelligent platform designed to connect people in need with the right support systems instantly.  
It helps users identify their needs, understand urgency, and discover the most relevant resources nearby.

---

## 🎯 Problem Statement
In real-world situations, people often struggle to find the right help quickly due to lack of awareness, guidance, and accessibility.  
Even when resources exist, users are unsure where to go or what to do next.

---

## 💡 Solution
CareConnect AI acts as a bridge between people in need and available support systems.  
It analyzes user input, classifies urgency, and recommends the most relevant help through an AI-powered system.

---

## ✨ Key Features
- AI-powered Chat Assistant for human-like guidance  
- Need & urgency classification system  
- Location-based help discovery  
- Personalized recommendations  
- Emergency detection and response  
- Impact dashboard for tracking requests  
- User profile and data management  

---

## 🛠️ Tech Stack
- **Frontend:** Streamlit  
- **Backend:** Python 
- **AI Models:** Gemini API, Groq (LLM)  
- **Database:** JSON-based storage  
- **Visualization:** Streamlit charts  

---

## ⚙️ How It Works
1. User logs in and describes their need  
2. System analyzes the request and detects urgency  
3. AI suggests relevant support options  
4. User can explore nearby resources  
5. Data is stored and visualized in the dashboard  

---
