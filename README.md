🚀 CareConnect AI – Connecting People to the Right Help, Instantly

CareConnect AI is an intelligent platform designed to help individuals quickly access essential support services such as medical aid, food assistance, financial help, and mental health resources using AI-driven interaction and location-based insights.

🌍 Problem

People in urgent situations often:

Don’t know where to find help
Waste time searching across platforms
Lack access to reliable, nearby services
💡 Solution

CareConnect AI provides:

Instant AI-based guidance
Smart categorization of needs
Nearby support services using location
A clean, user-friendly experience
🧠 Key Features
🤖 AI Chat Support (interactive assistance)
🗂️ Help Categories (Food, Medical, Financial, Mental Health, Education)
📍 Location-based services (Google Maps)
🔐 Secure Authentication
🎨 Modern, premium UI

⚙️ Setup Instructions
1. Clone the repo
git clone https://github.com/your-username/careconnect-ai.git
cd careconnect-ai
2. Setup Frontend
cd frontend
npm install
npm run dev
3. Environment Variables

Create a .env file inside frontend/

VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
VITE_GOOGLE_MAPS_API_KEY=your_maps_key
VITE_AI_API_KEY=your_ai_key
